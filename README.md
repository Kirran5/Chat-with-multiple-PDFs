# Chat with Multiple PDFs (RAG Application)

This project is a Retrieval-Augmented Generation (RAG) based application that allows users to upload multiple PDF documents and ask questions using Google Gemini and Hugging Face embeddings.

## Features
- Upload multiple PDFs
- Semantic search using FAISS
- Hugging Face embeddings
- Google Gemini for answer generation
- Streamlit UI
- CSV export of chat history

## Tech Stack
- Python
- Streamlit
- LangChain
- FAISS
- Google Gemini
- Hugging Face

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
